## 🚀 快速开始

如果您是第一次使用 FLUX Gateway，建议按以下顺序阅读：

1. **[项目介绍](./01-项目介绍.md)** - 了解项目的核心能力和架构设计
2. **[开发指南](./02-快速开始.md)** - 搭建开发环境并快速启动
3. **[安装部署](./03-安装部署.md)** - 部署到生产环境
4. **[容器化部署](./04-容器化部署.md)** - 使用 Docker/K8S 部署（可选）
5. **[数据库规范](./05-数据库规范.md)** - 了解数据库设计规范（开发人员必读）
